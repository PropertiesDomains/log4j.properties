About

Log4j.properties is a configuration file for the log4j logging framework.

Recently, '.properties' became a gTLD. This means that combined search / URL bars will no longer conduct searches for the filename, and instead route to http://log4j.properties/. This site was set up to help connect searchers find resources on log4j.
